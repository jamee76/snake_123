# Snake 🐍

Классическая игра «Змейка» на чистом HTML/CSS/JavaScript с использованием библиотеки [Kaplay.js](https://kaplayjs.com/).

## Как запустить

**Вариант 1 — Live Server (VS Code)**  
Откройте папку в VS Code и нажмите *Go Live* (расширение Live Server).

**Вариант 2 — Python**
```bash
python3 -m http.server 8080
# Откройте http://localhost:8080
```

**Вариант 3 — GitHub Pages**  
Включите GitHub Pages в настройках репозитория (Source: main / root).

> `index.html` сначала пробует загрузить `kaplay.js` из репозитория (для работы оффлайн),  
> при его отсутствии — обращается к CDN `unpkg.com`.

## Управление

| Клавиши | Действие |
|---------|----------|
| `←↑↓→` или `W A S D` | Поворот змейки |
| `Enter` / `Space` | Рестарт после Game Over |

## Структура кода (`game.js`)

| Блок | Назначение |
|------|-----------|
| `CONFIG` | Все параметры игры (размер поля, скорость, цвета) |
| `createGrid()` | Карта занятых клеток; точка расширения для стен/препятствий |
| `createSnake()` | Массив сегментов, направление, запрет 180° |
| `createFood()` | Спавн еды в случайной свободной клетке |
| `createGameState()` | FSM: `RUNNING` ↔ `GAME_OVER` |
| Scene `"game"` | Игровой цикл (тик + рисование) |
| Scene `"gameover"` | Экран конца игры со счётом и кнопкой Restart |
